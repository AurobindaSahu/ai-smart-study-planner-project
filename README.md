# ai-based-smart-study-planner
It is a full-stack AI-powered study planner using FastAPI and MySQL. Implemented a regression model to predict study hours and a scheduling engine to generate optimized study plans based on subject difficulty and deadlines.
The AI Smart Study Planner is a full-stack web application that helps students plan and manage their study schedules efficiently using Machine Learning.
It analyzes user input (subjects, deadlines, difficulty level) and generates a personalized study plan.
